## 🧩 Featured Projects

### Beacon — AR Navigation for Visually Impaired Users

An augmented reality navigation app that combines real-time object detection with voice guidance to help visually impaired users navigate safely. The critical challenge was optimizing computer vision models to run smoothly on mobile devices while maintaining accuracy for safe navigation—I implemented a priority system that focuses processing on obstacles and navigation-critical objects. Built with Flutter, Firebase, AR Kit, and OpenStreetMap, with emergency features that can send location data or call services directly.

[GitHub Repository](https://github.com/Shanu-Kumawat/Beacon)

---

### AgriSage — AI-Powered Agricultural Advisory

A precision agriculture app using machine learning to recommend optimal crops based on local weather patterns and soil conditions. The key architectural decision was running TensorFlow Lite models on-device rather than server-side, ensuring the app works in rural areas with limited connectivity—a critical requirement for farmers. Integrates Google Maps Engine for location services and provides real-time weather forecasting with satellite imagery.

### Quickshell Overview — Standalone Workspace Management
**Standalone Workspace Management Module**

When I switched Hyprland dotfiles from illogical-impulse to caelestia, I missed the workspace overview widget. Rather than accepting this limitation, I extracted it and refactored it into a standalone module by decoupling it from its original dependencies. This involved understanding Hyprland's socket protocol, restructuring the state management, and optimizing the update mechanism to refresh only when actual changes occur. Demonstrates how I approach problems—making useful tools more modular and accessible.

[GitHub Repository](https://github.com/Shanu-Kumawat/quickshell-overview)
